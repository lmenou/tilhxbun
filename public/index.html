A website done with htmx!

How to even use this ?

Simple search for an author on the input box below. If the Arxiv API find some articles published by the latter, the first three articles' titles will be displayed, clicking on them, will also display their abstract.

What is the point of all this ?

The idea is to test new techs with htmx/bun and co.
